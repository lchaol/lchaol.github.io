1.git add
git add -A (git add .)

2.git commit
git commit -m "本次提交的修改的备注"
新创建的文件要按顺序提交，修改可以用git commit -am "本次提交的修改备注"

3.git push
第一次提交到本分支
git push origin master(可替换其他分支名)
origin是远程仓库的默认名称，master是我们的分支名称(主分支)

第2-n次提交到本分支
git push 

提交到其他分支
git push origin b
提交到b分支
